2/2/14: Code Snippets for Search & Sort

Binary search and mergesort may be two of the most common techniques when it comes to searching or sorting some list of elements. Here is my implementation of them; of course some optimizations could be made (iterative vs. recursive, in-place vs. allocate space): 

    private static void merge(int[] arr, int lo, int mid, int hi) {
        int i = lo;
        int j = mid+1;
        int curr = lo;
        int[] helper = Arrays.copyOfRange(arr, 0, arr.length);
        while (i <= mid && j <= hi) {
            if (helper[i] <= helper[j]) {
                arr[curr] = helper[i];
                i++;
            } else {
                arr[curr] = helper[j];
                j++;
            }
            curr++;
        }
        for ( ; j < hi; j++) {
            arr[curr] = helper[j];
            curr++;
        }
        for ( ; i <= mid; i++) {
            arr[curr] = helper[i];
            curr++;
        }
    }

    private static void mergeSort(int[] arr, int lo, int hi) {
        if (lo < hi) {
            int mid = (hi+lo)/2;
            mergeSort(arr, lo, mid);
            mergeSort(arr, mid+1, hi);
            merge(arr, lo, mid, hi);
        }
    }

    private static void mergeSort(int[] arr) {
        mergeSort(arr, 0, arr.length-1);
    }

2/1/14: Longest Increasing Subsequence

While browsing StackOverflow, I came across an interview question where someone wanted an implementation of the longest increasing subsequence problem. I decided to do a quick code-up of the DP algorithm for it in Java. 

private static int[] longestIncrSub(int[] arr) {
        int[] solution = new int[arr.length];

        for (int i = 0; i < solution.length; i++) {
            int maxSeq = 0;
            for (int j = 0; j < i; j++) {
                if (arr[j] <= arr[i] && solution[j] >= maxSeq) {
                    maxSeq = solution[j];
                }
            }
            solution[i] = maxSeq + 1;
        }
        return solution;
}
And binary search: 
    private static int binSearch(int[] arr, int key) {
        int lo = 0;
        int hi = arr.length-1;

        while (lo <= hi) {
            int mid = (hi+lo)/2;
            if (arr[mid] == key) {
                return mid;
            } else if (arr[mid] > key) {
                hi = mid;
            } else {
                lo = mid+1;
            }
        }
        return -1;
    }
