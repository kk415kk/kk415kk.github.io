2/1/14: Longest Increasing Subsequence

While browsing StackOverflow, I came across an interview question where someone wanted an implementation of the longest increasing subsequence problem. I decided to do a quick code-up of the DP algorithm for it in Java. 

			private static int[] longestIncrSub(int[] arr) {
			        int[] solution = new int[arr.length];
			
			        for (int i = 0; i < solution.length; i++) {
			            int maxSeq = 0;
			            for (int j = 0; j < i; j++) {
			                if (arr[j] <= arr[i] && solution[j] >= maxSeq) {
			                    maxSeq = solution[j];
			                }
			            }
			            solution[i] = maxSeq + 1;
			        }
			        return solution;
			}
